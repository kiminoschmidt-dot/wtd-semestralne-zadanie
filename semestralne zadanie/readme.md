# WanderWorld - Cestovateľský portál

## Popis projektu
WanderWorld je cestovateľský webový portál vytvorený ako semestrálna práca pre predmet Webové technológie.

## Technológie
- HTML5
- CSS3
- JavaScript
- Lightbox2 (galéria)
- AJAX / Fetch API (načítanie dát z JSON)

## Štruktúra projektu

## Ako spustiť projekt
1. Stiahni alebo skopiruj repozitár z GitHub
2. Otvor priečinok projektu vo WebStorme
3. Klikni pravým tlačidlom na `index.html`
4. Vyber `Open in → Browser`
5. Web sa otvorí v prehliadači
